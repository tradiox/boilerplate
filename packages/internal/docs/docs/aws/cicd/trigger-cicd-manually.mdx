# Trigger CI/CD deployment manually

You can also trigger a deployment manually using GitHub Actions.

### Run GitHub Actions manually

If you have [GitHub Actions configured](./auto-deploy) in your repository, you can trigger a deployment manually.

1. Go to the GitHub repository and navigate to the "Actions" tab.

2. Select `Deploy to {ENVIRONMENT}` workflow.

3. Press the "Run workflow" button.

:::info
Learn how to run GitHub Actions manually from the
[documentation](https://docs.github.com/en/actions/managing-workflow-runs-and-deployments/managing-workflow-runs/manually-running-a-workflow).
:::
